Proyectos

Gym Contador de Series

Esta aplicación ayuda a los deportistas a facilitar el conteo de los diferentes ejercicios que realizan durante su entrenamiento. El usuario puede definir el número de series a realizar y el tiempo de descanso que desea tener entre cada una de las series. Cuando se culmina el tiempo de descanso el usuario podrá visualizar una alerta visual y auditiva que le indicará que debe retomar el entrenamiento.
Esta aplicación se hizo con HTML, CSS, JS y Sweet Alert.

Mokepon

Este aplicativo es un juego donde el usuario puede elegir su mascota entre 3 opciones (Hipodoge, Capipepo y Ratigueya) y su contrincante la computadora lo hará de manera aleatoria. Posterior a la elección de la mascota el usuario y su contrincante tienen 3 elementos (agua, fuego y tierra) para hacer sus 5 ataques, ganando quien obtenga mas puntos.
Este juego se hizo con HTML, CSS y JS.
